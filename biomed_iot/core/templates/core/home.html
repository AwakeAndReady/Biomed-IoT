Get started with Biomed IoT

Biomed IoT is an open-source IoT platform for collecting and storing sensor data, and automating devices.

Best for beginners:
The Manual demonstrates the platform with a simple use case.
How to use
  1. Register and log in
  2. Device List: Manage your IoT-device's MQTT-credentials
  3. Message & Topic Structure: Learn about message data format and mandatory topic structure
  4. Code Examples: Optionally copy and paste code templates for your IoT devices
  5. Automate: Create automation logic and store data into the time series database (InfluxDB) with Node-RED
  6. Visualize: Visualize data with Grafana
  7. Manage Data: Delete or export data as CSV-file
